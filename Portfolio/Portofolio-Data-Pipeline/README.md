# 카드 기반 경영 게임 데이터 설계 및 관리 시스템

## 1. 프로젝트 개요
본 프로젝트는 카드 기반 경영 게임에서 사용되는 게임 데이터를  
기획 변경과 콘텐츠 확장에 유연하게 대응할 수 있도록 설계한  
**데이터 설계 및 관리 시스템**입니다.

기획 단계에서 작성된 데이터를 Google Sheet에서 관리하고,  
이를 JSON 중간 포맷을 거쳐 Unity 런타임 데이터 구조로 변환하는  
데이터 파이프라인을 구축하여,  
**데이터 변경이 코드 수정 없이 런타임에 안정적으로 반영되도록** 구성했습니다.

---

## 2. 프로젝트 정보
- **프로젝트 명**: 카드 기반 경영 게임 데이터 설계 및 관리 시스템 구축
- **목표**: 기획 변경과 콘텐츠 확장에 유연하게 대응 가능한 데이터 파이프라인 구축
- **담당 역할**
  - 데이터 구조 설계
  - 데이터 파이프라인 구축
  - 런타임 연동 시스템 구현

> 본 저장소는 전체 프로젝트 중  
> **데이터 설계 및 관리 시스템과 관련된 핵심 코드만을 선별하여 공개한 포트폴리오용 Repository**입니다.  
> 실제 게임 프로젝트는 Private Repository로 관리 중입니다.
