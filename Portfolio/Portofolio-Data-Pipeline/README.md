# 데이터 설계 및 관리 시스템 (포트폴리오 코드)

본 저장소는  
**카드 기반 경영 게임 데이터 설계 및 관리 시스템 구축 프로젝트** 중  
데이터 파이프라인 및 런타임 데이터 관리 구조를 정리한  
포트폴리오 공개용 Repository입니다.

본 프로젝트의 전체 설계 배경, 문제 해결 과정, 결과에 대한 설명은  
**PDF 포트폴리오 문서에 상세히 정리되어 있습니다.**

---

## 포함된 코드 범위
- Google Sheet → JSON → Unity 런타임 데이터로 이어지는 데이터 파이프라인 핵심 로직
- JSON 파싱 및 런타임 데이터 매핑 구조
- 데이터 타입 확장을 고려한 자동 등록 및 관리 로직
- Unity Localization 기반 비동기 문자열 로딩 및 UI 바인딩 구조 구현

---

## 제외된 내용
- 실제 게임 콘텐츠 데이터
- UI 프리팹 및 연출 관련 코드
- 전체 프로젝트 코드 (Private Repository로 관리)
