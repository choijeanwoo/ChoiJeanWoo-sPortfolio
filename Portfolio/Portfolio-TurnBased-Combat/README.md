# SRPG 턴제 전투 시스템 (포트폴리오 코드)

이 폴더는  
**SRPG 턴제 전략 게임 전투 시스템 구현 프로젝트** 중  
전투 흐름, 이동·행동 판정, 상태 변화 처리를 담당하는  
**핵심 전투 로직 구조**를 정리한 포트폴리오 공개용 Repository입니다.

프로젝트의 전체 기획 배경과 설계 의도, 문제 해결 과정은  
PDF 포트폴리오 문서에 상세히 정리되어 있으며,  
이 폴더는 해당 설명을 뒷받침하는 코드 증빙을 목적으로 합니다.

---
## 포함된 코드 범위
- 턴 흐름 제어 및 전투 상태 관리 로직
- 이동·행동 가능 여부 판정 로직
- 명령 기반 전투 처리 구조

---
## 제외된 코드 범위
다음 코드는 전투 시스템의 핵심 구조보다는  
프로젝트 환경 및 입력 방식에 의존도가 높은 구현부이므로,  
이 폴더에서는 제외했습니다.

- 오브젝트 생성시 Template을 활용해 데이터를 반영하는 코드
- EventSystem을 통한 마우스 입력 처리 코드
- Command 패턴을 등록 또는 생성하는 실제 코드

---

## 참고
- 이 폴더는 포트폴리오 공개 목적의 요약 코드입니다.
- 실제 게임 콘텐츠 데이터 및 전체 구현부는 포함되어 있지 않으며,  
  전체 프로젝트는 Private Repository로 관리 중입니다.
